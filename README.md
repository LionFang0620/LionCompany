# LionCompany
## 專案簡介
模擬線上超市購物平台，包含會員端與後台管理端，完整實現商品管理與訂單處理流程。

## 功能列表
### 會員端
- 會員註冊與登入
- 商品瀏覽與搜尋
- 加入購物車、下單與結帳
- 查看訂單紀錄

### 後台管理端
- 新增、修改商品分類與商品
- 調整商品價格
- 訂單管理與查詢

## 使用技術
- Java（JDK 17）
- JDBC
- MVC 架構
- JFrame GUI
- MySQL 資料庫

## 專案架構
src/
│── model/ # 資料物件類別
│── dao/ # 資料存取層
│── view/ # GUI 視窗介面
│── controller/ # 控制邏輯
sql/ # 初始化資料庫
lib/ # 外部依賴
screenshots/ # GUI截圖
